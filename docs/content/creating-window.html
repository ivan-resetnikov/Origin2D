Creating an empty window

Origin2D's structure is diffecret for your any other framework.
It takes more "game-engine-like" approach, like using abstract layers,
like scenes, actors etc.

Scene - is a "container" for something like a level,
scene can represent a main menu, settings menu, a game level etc.

Actor - is a component of a scene, actor can represent
a player, an enemy, a bullet, an item etc.

Creating a main scene

First let's create our main.py

main.py 

import core       # import Origin2D's core
import main_menu  # import your main scene



if __name__ == "__main__":
	main_scene = core.Scene(
		callback_init=main_menu.init,      # initialisation function, called when scene is loaded
		callback_update=main_menu.update,  # update function, called every frame
		callback_draw=main_menu.draw,      # draw function, called every frame
		name="main_scene")                 # scene name

	game = core.Game()             # game (game manager) instance, controlls the entire game
	game.change_scene(main_scene)  # Change scene to our main scene
	game.run()                     # Run the game

As a side note: It is important that you first,
load your scene, and only then you run your game.

Init function is being called when scene is loaded,
it's recomended to initialize your game components here. (Player, enemies, level objects etc.)

Update function is being called every frame, it is recomended
to run all update loginc here. (Moving a player when certain key is pressed, enemies logic, etc.)

Draw function is also being called every frame, it is recomended
to do all draw logic here. (Drawing player sprite, animating actors, etc.)
